## üéØ Program Overview

**Duration**: 24 weeks (part-time) or 12 weeks (intensive)
**Format**: Self-paced with optional cohort sessions
**Prerequisites**: Python proficiency, security fundamentals

### Learning Tracks

1. **Offensive AI** - Build AI-powered pentesting and red team tools
2. **DFIR AI** - Automate detection, response, and forensic analysis
3. **Threat Intelligence** - AI-driven threat intel processing and analysis
